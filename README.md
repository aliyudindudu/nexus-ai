## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm / bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aliyudindudu/nexus-ai.git
   cd nexus-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```
